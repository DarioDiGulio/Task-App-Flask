### Comandos

* Crear entorno virutal  
`python3 -m venv venv`
* Activar entorno virtual  
`source venv /bin/activate`
* Desactivar entorno virtual  
`deactivate`


### Dependencias
* Flask  
`pipenv install flask`
* SQLAlchemy  
`pipenv install Flask-SQLAchemy`
* Sqlite3  
`pipenv install pysqlite3`

### DB
* Crear base de datos   
`pysqlite3 database/task.db`
